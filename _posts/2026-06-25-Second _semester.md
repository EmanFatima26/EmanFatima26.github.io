# 🌸 Second Semester

My second semester was one of the most challenging and rewarding phases of my university journey. It was filled with new technologies, unforgettable experiences, health struggles, and many achievements.

---

<details>
<summary><h2>📖 Overview</h2></summary>

During this semester, I focused on improving my programming, database, and software development skills. My confidence grew significantly as I started working on real-world projects instead of only theoretical concepts.

</details>

---

<details>
<summary><h2>📚 Subjects Learned</h2></summary>

- Database Systems
- Object-Oriented Programming (C#)
- Digital Logic Design
- Calculus II

I learned SQL, database normalization, C# programming, and practical software development through labs and assignments.

</details>

---

<details>
<summary><h2>💻 Major Projects</h2></summary>

### GitHub Portfolio

I designed my first GitHub portfolio using Jekyll and GitHub Pages.

### Crime Prediction Project

A Machine Learning project that predicts crime arrest probability using FastAPI, MySQL, HTML, CSS, JavaScript, and Python.

### DLD Project

Built a Digital Counter using infrared sensors and electronic hardware.

</details>

<details>

This semester was not only academically difficult but also physically demanding.

Every time I returned from home to the hostel, I suffered from high fever and stomach problems. Despite these difficulties, I remained committed to my studies and completed every project successfully.

</details>

<details>
<summary><h2>🌙 Portfolio Night</h2></summary>

One of the most memorable nights of this semester was when Dr. Bilal assigned us to create a professional GitHub portfolio.

I worked until almost 6 AM, redesigning the website, fixing deployment errors, and learning GitHub Pages. Although it was exhausting, the final result made me proud.

</details>

<details>
